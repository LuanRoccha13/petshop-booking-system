# Image Assets Guide

Esta pasta centraliza as imagens do frontend.

## Estrutura

- landing: JPG/PNG usados na landing page
- shared: imagens reutilizaveis entre paginas
- auth: imagens de apoio para login e cadastro
- dashboard: imagens de apoio para dashboard
- illustrations: ilustrações e empty states

## Regras de organizacao

1. Nomeie arquivos com padrao secao-funcao-variacao.ext.
2. Prefira JPG para fotos e PNG para transparencias.
3. Nao usar espacos nem acentos no nome do arquivo.
4. Mantenha dimensoes coerentes com o uso em tela.
