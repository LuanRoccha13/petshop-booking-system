# 10 Antigravity Image Pass Prompt

Use este prompt para a rodada de integracao de imagens com continuidade do estilo atual.

PROMPT

Contexto importante:

O estilo da landing ja existe e deve ser preservado. Nao refaça a identidade visual. Esta rodada e para integrar imagens de forma inteligente e deixar o site mais vivo mantendo a qualidade do design atual.

Use obrigatoriamente as especificacoes:

- 01-brand-direction.md
- 02-design-tokens.md
- 03-motion-system.md
- 04-component-specs.md
- 05-page-recipes.md
- 07-accessibility-quality-gates.md
- 09-image-asset-system.md

Use imagens a partir desta estrutura:

- src/assets/images/landing
- src/assets/images/shared
- src/assets/images/auth
- src/assets/images/dashboard
- src/assets/images/illustrations

Objetivo da rodada:

1. Integrar imagens JPG e PNG nas paginas existentes.
2. Preservar o estilo atual e elevar qualidade visual.
3. Melhorar sensacao de produto vivo sem perder clareza.
4. Garantir responsividade completa com imagens.

Regras obrigatorias:

1. Nao mudar a direcao visual central da landing.
2. Nao usar imagens como decoracao gratuita; cada imagem precisa ter funcao.
3. Nao comprometer legibilidade de titulo, subtitulo e CTA.
4. Em telas internas, usar imagens de apoio com equilibrio para nao distrair do fluxo.
5. Dashboard deve ficar moderno, com imagens bem posicionadas e acabamento premium.

Escopo minimo:

1. Landing:
   - hero com imagem protagonista
   - servicos com imagens de apoio
   - depoimentos com imagem/avatares quando fizer sentido
2. Login/Cadastro:
   - painel visual lateral no desktop
   - comportamento simplificado no mobile
3. Dashboard:
   - imagem de contexto no topo ou lateral
   - cards de agendamento com imagem bem tratada
   - estado vazio com ilustração discreta

Responsividade:

1. Ajustar recortes por breakpoint sem distorcao.
2. Garantir boa proporcao e alinhamento em 320px, 768px e desktop.
3. Evitar blocos altos demais no mobile.

Formato de entrega:

1. Lista de arquivos alterados.
2. Resumo de onde cada grupo de imagem foi aplicado.
3. Checklist final de qualidade e responsividade.

PROMPT FIM
