# 06 Copy Voice

## Voice profile

- Proximo e acolhedor
- Profissional e objetivo
- Confiante sem exagero publicitario

## Writing principles

1. Frases curtas e claras.
2. Beneficio primeiro, detalhe depois.
3. Evitar termos tecnicos para o cliente final.
4. Sempre indicar proximo passo em mensagens de estado.

## Microcopy patterns

- Sucesso: Agendamento criado com sucesso.
- Erro de formulario: Revise os campos destacados.
- Erro de rede: Nao foi possivel concluir agora. Tente novamente.
- Acao destrutiva: Cancelar agendamento
- Confirmacao opcional: Tem certeza que deseja cancelar?

## CTA rules

- CTA primaria: verbo + beneficio
- Maximo 3 palavras na label
- Exemplo principal: Quero agendar
